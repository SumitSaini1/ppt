CareScan AI

Early & Accurate Detection of Parkinson's Disease

Using Multimodal AI Analysis of Speech and Handwriting


Project Report Overview

1. Motivation & Overview

The Challenge

Parkinson's Disease (PD) is a complex neurodegenerative disorder affecting millions globally. Early detection is crucial but difficult.

The Solution: CareScan AI

  • Multimodal Analysis: Combines Speech (vocal tremors) and Handwriting (motor impairments) analysis.
  • Non-Invasive: Users simply record voice or upload images of spirals/waves.
  • Doctor-Patient Interface: Secure portal for appointments, prescriptions, and report sharing.

2. Objectives

  • Robust AI Algorithms: Analyze user-uploaded voice and handwriting samples to predict PD risk accurately.
  • Secure Portal: Web-based platform for patients and healthcare providers.
  • Management Features: Facilitate appointment booking and collaborative consultations.
  • Real-time Reporting: Provide clear diagnostic reports with risk scores.
  • Ultimate Goal: Improve early detection rates and empower continuous patient care.

3. Key Modules

1. Speech Analysis

Uses Librosa to extract features like Jitter (tremor) and Shimmer (amplitude).

2. Handwriting Analysis

Uses OpenCV to analyze spirals/waves for tremors and irregularities.

3. Doctor Collaboration

Secure login for doctors to review reports and generate e-prescriptions.

4. Appointment System

Workflow for booking and managing patient-doctor interactions.

4. Technical Feasibility

  • Frontend: React / HTML / CSS / JavaScript
  • Backend Hub: Node.js (Express)
  • AI Microservices: Python Flask
  • Machine Learning: Librosa, OpenCV, Scikit-learn
  • Database: MongoDB (NoSQL)

5. Level 0 DFD (Context Diagram)

System Overview

The Level 0 DFD visualizes the system boundaries and high-level interactions:

  • User (Patient): Provides Voice and Handwriting samples. Receives Diagnostic Reports.
  • CareScan AI: Central processing unit acting as a "Black Box".
  • Doctor: Reviews AI-generated results and sends Prescriptions back to the system.

6. Level 1 DFD (Detailed Flow)

Process Breakdown

Inside the CareScan System boundary:

  • Auth Module: Handles Login/Signup using Node.js/Passport.
  • Record Management: Handles file uploads.
  • Diagnosis Processing: Routes data to specialized Flask services.
  • External Processing: Voice data goes to Librosa Service; Images go to OpenCV Service.

7. UML Class Diagram

Object Structure

  • User Class: The parent class containing shared attributes like ID and email.
  • Inheritance: 'Patient' and 'Doctor' inherit from User, adding specialized methods (e.g., uploadSample vs addPrescription).
  • Relationships: Patients 'Book' Appointments. Appointments 'Create' Diagnosis results.

8. UML Sequence Diagram

Interaction Flow

Timeline of a Diagnosis:

  1. User uploads a file via Frontend.
  2. Frontend hits Node.js API endpoint.
  3. Node.js forwards file to Flask Engine.
  4. Flask processes ML logic and returns JSON.
  5. Node.js saves results to MongoDB.
  6. System displays report to User.

9. System Architecture

Hub-and-Spoke Model

The architecture is designed for scalability:

  • Node.js Orchestrator (Hub): Central manager. Handles authentication and routing.
  • Frontend (Spoke): Lightweight React UI.
  • Flask Engines (Spokes): Dedicated microservices for heavy AI computation (Voice/Image).
  • MongoDB (Data): Persistent storage connected to the Hub.

10. Entity-Relationship (ER) Diagram

Database Relationships

A logical view of data connectivity:

  • User Centrality: The User entity is the core, connecting to all other data points.
  • One-to-Many Relationships:
    • 1 User has Many Appointments.
    • 1 User has Many Voice Diagnosis results.
    • 1 User has Many Handwriting results.

11. Database Collections

Users: Authentication (ID, email, hashed password, role).

Appointments: Date, time, status, Doctor_ID, Patient_ID.

Voice_Diagnosis: Stores Jitter, Shimmer, HNR values, and Risk Score.

Handwriting_Diagnosis: Stores tremor index, roughness, and Risk Score.

Conclusion

Impact

CareScan AI provides a feasible, operational, and technically robust solution for the early detection of Parkinson's Disease.


Future Scope

Integration with wearable sensors, expansion to other neurodegenerative disorders, and mobile app development.
